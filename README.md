# Getting Started

## Requirements:

```
Maven: 3.9+
JDK: 23
```

### Clone this repository:

```bash
git clone https://github.com/deepaksorthiya/java-interview-maven-project.git
```

```bash
cd java-interview-maven-project
```

### Run Test(Maven Should Be Installed):

```bash
mvn clean test
```
